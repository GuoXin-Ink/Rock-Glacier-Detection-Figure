# Rock-Glacier-Detection-Figure

This repository contains all the Generic Mapping Tools (GMT) scripts used to generate the figures for the study: "Mapping Rock Glaciers in the Southeastern Tibetan Plateau Using Machine Learning: Advancing a Comprehensive Inventory and Accuracy Assessment".

## Overview

- **Purpose**: Reproduction of all figures from the manuscript.
- **Tool**: All scripts are written for **Generic Mapping Tools (GMT)**.
- **Data**: The scripts use processed/result data from the main analysis code (see Related manuscript).

## Requirements

- **Generic Mapping Tools (GMT)** version 6.0 or later
- **Bash** shell environment (Linux/macOS) or WSL (Windows)
